Memoraid: A Deep Learning Based System for Early Detection of Dementia

Memoraid is a deep learning-based application developed to detect and classify dementia stages using MRI brain images.
The project uses ResNet-18 with transfer learning to analyze brain scans and provide prediction results through a simple Streamlit interface.

Tech Stack

Python, PyTorch, CNN, ResNet-18, Streamlit, OpenCV

Steps to Run the Project

1. Create Virtual Environment
   python -m venv venv
   
3. Activate the Environment
   venv\Scripts\activate
   
5. Install Required Dependencies
   pip install -r requirements.txt
   
7. Run the Application
   streamlit run app.py
